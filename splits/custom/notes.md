### For compound exercises
3 increasingly heavy working sets
5-12 reps max
Rest 2 min between set and exercises
superset = 6 rep + 6 rep

### For isolation exercises
Do 1 warmup set (30% of 1 rep max) for 10 reps
Working set = (60% of 1 rep max)
Do 1 working set to failure or beyond (negatives) and drop sets and again failure and negatives until half working set weight
Rest 1 min between exercises
superset = failure + failure
